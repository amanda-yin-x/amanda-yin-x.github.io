# Amanda Yin - Personal Portfolio

A personal portfolio website featuring an interactive paper crane intro animation and a handwritten letter aesthetic.

## Live Site

[amanda-yin-x.github.io](https://amanda-yin-x.github.io/)

## Features

- **Paper Crane Intro** - Interactive origami crane that unfolds as you scroll, revealing the portfolio content
- **Handwritten Aesthetic** - Warm parchment tones, ink colors, and handwritten typography
- **Responsive Design** - Clean hamburger navigation on all screen sizes
- **Smooth Animations** - Framer Motion powered transitions throughout

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## Pages

- `/` - Home with paper crane intro
- `/experience` - Work experience and roles
- `/skills` - Technical skills and tools
- `/entrepreneurship` - Ventures and projects
- `/contact` - Contact form and links

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Design

The design draws inspiration from:
- Traditional origami paper cranes
- Handwritten letters and stationery
- Warm parchment and ink aesthetics
- Tiffany blue accent color (#0ABAB5)
